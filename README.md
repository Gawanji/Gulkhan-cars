# Car Maintenance Website

A responsive car maintenance and repair service website template.

## Features

- **Modern Design**: Clean and professional user interface.
- **Service Listings**: Detailed information about various car services.
- **Booking System**: Integrated WhatsApp booking functionality.
    - Click "Book Now" to automatically open a WhatsApp chat with pre-filled booking details.
- **Responsive**: Works great on distinct devices (Desktop, Tablet, Mobile).

## How to Run

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Gawanji/Gulkhan-cars.git
    cd Gulkhan-cars
    ```

2.  **Start a local server:**
    You can use Python's built-in HTTP server:
    ```bash
    python -m http.server 8000
    ```

3.  **Open in Browser:**
    Navigate to `http://localhost:8000` to view the website.

## Technologies Used

- HTML5
- CSS3 (Bootstrap 5)
- JavaScript (jQuery)
- Font Awesome
